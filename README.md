# Run Instructions

## Setup your environment 
You will need to have in your system

- Java 17.0 or higher
- Node and npm
- Apache Maven
- IDE or Editor
- Docker

## Full Application 
-cd into main folder where the frontend and backend directories are. 
run docker-compose build 	
    docker-compose up 

Browse the application from localhost:3000.
